Build your own native mobile apps for your WordPress site with [App La Carte](http://www.applacarte.com/).

== Description ==

Create a completely native mobile phone app with App La Carte by simply installing a plugin, absolutely no coding necessary! Once installed visit [App La Carte](http://www.applacarte.com/ "Apps in a click") set up your app, enter your website URL and you're done.  

= Setting up =

Install the App La Carte plugin, set up your app by visiting [App La Carte](http://www.applacarte.com/ "Apps in a click"). Select Wordpress as a feature, choose your theme colors and fonts, add in your website URL in the Wordpress tab and you're done. To view your app on a mobile phone download the App La Carte app available on iTunes or the Google Play store and enter your login details.

= Features =

This plugin creates a full featured native app for your iPhone and Android! With a fantastic interface and a whole lot of features built right in including Search, viewing and posting comments, viewing image galleries. You can be confident your website readership will shoot through the roof.

== Installation ==

1. Upload the 'applacarte_wp' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to App La Carte page under Settings section.
4. Check the categories you want to display on the mobile app. 
3. Visit [www.applacarte.com](http://www.applacarte.com) to create and customize your app.
